# IdiTusi API

## Getting started

```shell
  make run
```

## Minimum Requirements

|        | Version                                 |
|--------|-----------------------------------------|
| Go     | 1.21                                    |
| Docker | Docker Engine 24.00, Docker Compose 3.9 |
| Make   | GNU Make 4.4.1                          |

## Tech Stack

|       | Description      |
|-------|------------------|
| Fiber | HTTP framework   |
| PGX   | Postgres toolkit |
